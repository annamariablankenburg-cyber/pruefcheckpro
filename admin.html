Stripe Setup

Verbinde dein Stripe-Konto für Zahlungen.

Schritt 1 — Stripe Account
Registriere dich kostenlos auf stripe.com. Kein Kreditkarte nötig — zahlen tust du erst wenn Kunden zahlen (1,4% + 25ct pro Transaktion).
stripe.com öffnen →
Schritt 2 — API Keys
In Stripe unter Developers → API Keys deinen Publishable Key (pk_live_...) hier eintragen.
Schritt 3 — Payment Links
In Stripe unter Products → Payment Links je einen Link für Pro (49€) und Enterprise (99€) anlegen. Die Links hier eintragen.
Schritt 4 — Speichern & Testen
Nach dem Speichern erscheinen die Stripe-Links automatisch auf der Preisseite der Website und in der App.
